# 🌊 Flood Alert Monitoring System

A responsive multi-page web application that predicts flood risk using real-time weather data.

## 🚀 Features
- City-based flood risk detection
- Color-coded risk indicator
- Dynamic progress bar
- Weather-based icon display
- API integration (OpenWeatherMap)
- Responsive UI design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- GitHub Pages
